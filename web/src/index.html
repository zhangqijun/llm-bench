🎯 快速开始

  1. 启动 vLLM 服务：python -m vllm.entrypoints.openai.api_server --model Qwen/Qwen2.5-0.5B-Instruct
  2. 启动测试工具：cd web && ./start_tps_test.sh
  3. 访问测试页面：http://localhost:8080/tps-test.html
  4. 配置 API URL 为 http://localhost:5000/v1（使用代理）或 http://localhost:8000/v1（直连）
  5. 点击"开始测试"按钮，查看实时 TPS 性能
